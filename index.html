Projects

I have been working on creating and testing different environments for interaction biasing. Interaction biasing is the process of influencing a robot to take different paths depending on external factors. Robots have the ability to interact with each other and I am writing test cases and creating environments that show that we have the ability to program a robots to operate aroundthese interactions. For example, if one of the interactions involves reaching a desitnation that goes through a storm, then the route would be changed to avoid that storm. We are able to do this by changing the weight of certain edges in the road map to prevent a robot from going in that direction. The various environments that I have created are designed in such a way that having an interaction between two robots is most ideal. But threre are alternate routes in the environmnet that don't require those interactions that we are going to bias a robot into taking.

My objective is to create environmnets that show that the bias causes a robot to take a more difficult path when the original interaction would have been more direct.

I spent the entire day creating a large environment (50x100) that shows how we can navigate around complicated terrain. The way that the environment is designed is that if a land robot performs an interaction with a water robot it will result in many narrow passages but if it initially takes a longer path by itself it will get to the goal configuration with ease.

The idea behind all of these environments is that there are multiple paths to take and we want to put forward a bias to take the most ideal route whether it is because of a shortage of fuel or to avoid difficult terrain. All of the environments are built so that there is a path that they will take and a different path that they can take and we are going to impose bias to take the path that they aren't taking.
